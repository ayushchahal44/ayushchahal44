Full Stack Developer
--------------------

Hi, I’m a software engineer with a strong foundation in web development and a range of experiences across different industries. My expertise lies in technologies like JavaScript, TypeScript, React, Node.js, Next.js, Docker, Kubernetes, MongoDB, and MySQL. Throughout my career, I’ve worked on key projects focused on SEO optimization, performance enhancement, and UX improvements. I’m particularly skilled at reducing the Largest Contentful Paint (LCP) to boost user experiences, and I’ve implemented various strategies like lazy loading, asynchronous loading, and compression techniques to optimize JavaScript payloads. I also developed a dynamic design rendering engine and worked on middleware services, integrating transaction functionalities across multiple endpoints. In a previous role at Leave Code Technologies Pvt. Ltd., I developed a full-featured examination portal using the MERN stack. This portal supports online exam creation, management, and administration, with secure JWT-based user authentication and authorization. Working with MongoDB, I ensured scalable and efficient data handling, while my React.js skills enabled me to build engaging and interactive UI components. I’ve also created a real-time coding collaboration app with the MERN stack and WebSocket technology, as well as an electronic voting system in C—showcasing my flexibility across different platforms and languages. Recently, I completed a virtual internship with CodSoft, where I worked on projects such as a React-based calculator, a MERN stack website, and a personal portfolio. I also built a weather dashboard that integrates data from OpenWeatherMap and RapidAPI for geolocation, using Material-UI to ensure a responsive, visually appealing design. This dashboard provides current weather, a 7-day forecast, and data visualizations for temperature trends, making it both user-friendly and informative across devices. I’m passionate about technology and driven to deliver high-quality, user-focused solutions. With a wide skill set in both frontend and backend technologies, I bring technical expertise and creative problem-solving to every project.
